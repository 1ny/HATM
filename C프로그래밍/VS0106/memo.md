### static 변수
클래스 내에서만 쓰는 static 변수, 즉 클래스 변수도 있다
```cs

class Fishbun {
public:
	int cnt1;
	static int cnt2;

	Fishbun():cnt1{ 0 } {
		++cnt1;
		++cnt2;
	}
};

int Fishbun::cnt2 = 0;	// static 변수 초기화 방법

int main() {
	Fishbun f1;
	std::cout << f1.cnt1 << "," << Fishbun::cnt2 << std::endl;
	Fishbun f2;
	std::cout << f1.cnt1 << "," << Fishbun::cnt2 << std::endl;
	Fishbun f3;
	std::cout << f1.cnt1 << "," << Fishbun::cnt2 << std::endl;


	return (0);
}
```
멤버 변수는 인스턴스마다 독립적이지만, 멤버함수는 모든 인스턴스가 공유한다.

### 함수 오버로딩 vs 오버라이딩
오버라이딩은 상속받은 클래스 함수를 자식 클래스에서 변경하는 것.  
오버로딩은 함수명은 같지만, 파라미터의 개수나 타입을 다르게 하여 재정의하는 것.  
C++에서는 오버로딩이 가능하다.  
다만, 반환 타입으로는 구별이 안된다. 파라미터의 타입이나 개수로 구분해야한다.  
```cs
/* overriding example */
class Father {
public:
	void draw() {
		cout << "Father" << endl;
	}
}

class Daughter:public Father {
public:
	void draw() {
		cout << "Daughter" << endl;
	}
}
```
```cs
/* overloading example */
class Father {
public:
	void draw() {
		cout << "Father" << endl;
	}
	void draw(string a) {
		cout << "Father " + a << endl;
	}
}
```
